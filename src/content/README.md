# Content Replacement Guide

Replace the placeholder portfolio content in `src/data`:

- `profile.ts` for personal positioning, links, email, and resume URL.
- `projects.ts` for case study content, metrics, GitHub links, live demos, and architecture notes.
- `skills.ts` for skill levels, categories, and tools.
- `experience.ts` for work history and interview-facing career context.

The Vue sections consume typed data, so most content updates do not require editing component templates.
